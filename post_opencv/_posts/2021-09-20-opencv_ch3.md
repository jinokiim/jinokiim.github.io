### 이미지 생성


```python
import cv2
import numpy as np

img = np.zeros((120,120), dtype=np.uint8)
img[25:35, :] = 45
img[55:65, :] = 115
img[85:95, :] = 160
img[:, 34:45] = 205
img[:, 75:85] = 255
```

### subplot


```python
import matplotlib.pyplot as plt
import numpy as np

x=np.arange(10)

plt.subplot(2,2,1)
plt.plot(x, x**2)

plt.subplot(2,2,2)
plt.plot(x,x*5)

plt.subplot(223)
plt.plot(x, np.sin(x))

plt.subplot(224)
plt.plot(x,np.cos(x))

plt.show()
```
